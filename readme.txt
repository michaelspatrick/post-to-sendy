Automatically schedule and send your WordPress posts to a Sendy email list.

== Description ==

Post to Sendy allows WordPress site owners to integrate with Sendy and schedule posts for email delivery at optimal times or via a custom schedule.

== Features ==

* Choose between Never, Optimal, or Custom schedule per post
* Send posts to Sendy email list automatically
* Admin settings page for Sendy API configuration
* Built for security with WordPress standards

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/post-to-sendy` directory, or install via the WordPress plugins screen.
2. Activate the plugin through the ‘Plugins’ screen in WordPress.
3. Navigate to *Settings > Post to Sendy* to configure your Sendy API.
4. Use the metabox in your post editor to schedule email delivery.

== Frequently Asked Questions ==

= What is Sendy? =
Sendy is a self-hosted email newsletter application that lets you send trackable emails via Amazon SES.

= Can I disable sending on a per-post basis? =
Yes, simply select "Never" in the post metabox.
